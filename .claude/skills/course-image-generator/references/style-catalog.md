# Summary Filmina Style Catalog

Complete reference for the 8 rotating summary styles used in ZoneKlass lesson summaries.

## Style Rotation System

Styles rotate by lesson index: `lessonIndex % 8`

Example for a course with 42 lessons:
- Lessons 0, 8, 16, 24, 32, 40 → **Chalkboard**
- Lessons 1, 9, 17, 25, 33, 41 → **Modern Infographic**
- Lessons 2, 10, 18, 26, 34 → **Isometric**
- ...and so on

This ensures visual variety throughout the course while maintaining a consistent pattern.

---

## Style 0: Chalkboard

### Visual Description
A realistic dark green classroom chalkboard with authentic chalk-drawn elements.

### Key Characteristics
- **Background:** Dark green chalkboard texture with visible chalk dust and smudges
- **Text:** White chalk for main text, colored chalk for highlights
- **Elements:** Hand-drawn educational icons (lightbulb, book, brain, gear, chart)
- **Accents:** Module-specific accent color for highlights
- **Details:** Chalk borders, eraser marks, arrows connecting concepts

### Color Guidance
- Primary: White chalk (#FFFFFF)
- Highlight: Module accent color in chalk style
- Background: Dark green (#1A4D2E to #2D5F3F)
- Shadows: Subtle smudging effect

### Mood
Nostalgic, academic, traditional classroom, warm and familiar

### Tips for Best Results
- Emphasize hand-drawn quality over perfect lines
- Include chalk texture and dust particles
- Use arrows and connecting lines between concepts
- Add subtle eraser marks for authenticity

### Example Use Cases
- Fundamental concepts that benefit from traditional teaching aesthetic
- Lessons on basic principles or foundations
- When evoking classic educational experience

## Style 1: Modern Infographic

### Visual Description
Clean, professional infographic with flat design elements and minimal aesthetic.

### Key Characteristics
- **Background:** Clean white or light gray (#F9FAFB)
- **Icons:** Flat design, geometric, minimalist
- **Layout:** Numbered sections with connecting lines or flowchart
- **Typography:** Clean sans-serif, professional hierarchy
- **Style:** Similar to Notion, Stripe, or modern SaaS landing pages

### Color Guidance
- Primary: Module colors (indigo, emerald, coral, etc.)
- Accent: Module accent color for emphasis
- Background: White or very light gray
- Text: Dark gray (#1F2937) or black

### Mood
Professional, modern, clean, tech-forward, trustworthy

### Tips for Best Results
- Use ample white space
- Align elements precisely in grid layout
- Keep icons simple and geometric
- Use subtle shadows only
- Emphasize data visualization aesthetic

### Example Use Cases
- Business or professional topics
- Technical concepts requiring clarity
- Lessons on modern tools/platforms
- Step-by-step processes

## Style 2: Isometric

### Visual Description
3D isometric perspective with floating platforms and interactive characters.

### Key Characteristics
- **Perspective:** 30-degree isometric angle
- **Elements:** 3D blocks, platforms, floating objects
- **Characters:** Small figures interacting with concepts
- **Layout:** Connected platforms showing concept flow
- **Depth:** Soft shadows and layered elements

### Color Guidance
- Primary: Module colors with depth
- Shadows: Soft gray (#9CA3AF) at 30% opacity
- Highlights: Module accent for emphasis
- Background: Light gradient or solid white

### Mood
Playful, modern, tech-forward, engaging, game-like

### Tips for Best Results
- Maintain consistent 30-degree angle
- Use characters to show scale and interaction
- Connect platforms with paths or arrows
- Add subtle gradients for depth
- Keep colors vibrant but cohesive

### Example Use Cases
- System architecture or workflows
- Multi-step processes
- Platform overviews
- Lessons with multiple interconnected concepts

## Style 3: Watercolor

### Visual Description
Beautiful watercolor painting with soft brush strokes and artistic elegance.

### Key Characteristics
- **Medium:** Watercolor painting aesthetic
- **Texture:** Visible brush strokes, paint bleeds, organic shapes
- **Background:** Cream or off-white textured paper
- **Typography:** Hand-lettered calligraphy in watercolor style
- **Icons:** Painted vignettes arranged artistically

### Color Guidance
- Primary: Module colors with watercolor transparency
- Splashes: Module accent as color bursts
- Background: Cream (#FFFBF0) or light beige
- Bleeds: Soft color transitions between concepts

### Mood
Artistic, inspiring, creative, elegant, dreamy, sophisticated

### Tips for Best Results
- Embrace organic, flowing shapes
- Show paint bleeds between colors
- Use visible brush texture
- Add water droplet effects
- Keep composition balanced but not rigid

### Example Use Cases
- Creative or artistic topics
- Design lessons
- Soft skills or personal development
- Lessons requiring inspirational feel

## Style 4: Comic/Caricature

### Visual Description
Fun comic book style with cartoon characters and speech bubbles.

### Key Characteristics
- **Style:** Bold outlines, halftone dots, action lines
- **Characters:** Friendly teacher/robot/owl mascot explaining concepts
- **Layout:** Comic panels showing each concept as mini-scene
- **Effects:** POW/ZAP graphics, motion lines, expressive characters
- **Mood:** Educational but entertaining

### Color Guidance
- Primary: Bright, saturated module colors
- Outlines: Black (#000000) at 3-4px width
- Halftone: Module accent in dot patterns
- Highlights: White with comic book gleam

### Mood
Fun, engaging, youthful, energetic, approachable

### Tips for Best Results
- Use bold black outlines on all elements
- Add motion lines for energy
- Include expressive character faces
- Use speech/thought bubbles for concepts
- Add comic-style sound effects sparingly

### Example Use Cases
- Beginner-friendly lessons
- Complex topics needing approachability
- Courses for younger audiences
- Lessons benefiting from humor/personality

## Style 5: Neon/Dark

### Visual Description
Futuristic cyberpunk aesthetic with glowing neon elements on dark background.

### Key Characteristics
- **Background:** Deep dark navy or black (#0F172A)
- **Neon:** Glowing outlines in cyan, magenta, module accent
- **Interface:** Holographic HUD-style elements
- **Grid:** Subtle background grid pattern
- **Effects:** Light particles, glowing orbs, energy lines

### Color Guidance
- Background: Dark navy (#0F172A) to black (#000000)
- Neon Primary: Cyan (#06B6D4), Magenta (#EC4899)
- Accent: Module accent in neon glow
- Highlights: White glow (#FFFFFF) at 80% opacity

### Mood
Futuristic, cutting-edge, high-tech, cyberpunk, innovative

### Tips for Best Results
- Use outer glow effects on all neon elements
- Add subtle grid in background
- Include floating particles
- Make text glow intensely
- Use holographic/translucent overlays

### Example Use Cases
- AI and machine learning topics
- Future tech or innovation
- Advanced technical concepts
- Lessons on emerging technologies

## Style 6: Vintage Notebook

### Visual Description
Aged student notebook with handwritten notes and sketched diagrams.

### Key Characteristics
- **Background:** Yellowed lined notebook paper
- **Text:** Handwritten fountain pen style in dark ink
- **Elements:** Paper clips, sticky notes, washi tape
- **Highlights:** Module accent as highlighter marks
- **Details:** Coffee rings, page wear, margin doodles

### Color Guidance
- Paper: Aged cream (#FFF8DC) to light beige
- Ink: Dark brown-black (#2C1810)
- Highlights: Module accent at 40% opacity
- Accents: Sticky notes in pastel colors

### Mood
Nostalgic, studious, personal, cozy, authentic

### Tips for Best Results
- Add imperfections (coffee stains, creases)
- Use varied handwriting for emphasis
- Include margin sketches and doodles
- Add sticky notes for key terms
- Show page texture and wear

### Example Use Cases
- Note-taking or study skills lessons
- Historical or retrospective topics
- Personal development or journaling
- Lessons emphasizing individual learning

## Style 7: 3D Render

### Visual Description
Smooth, glossy 3D rendered scene with soft lighting and toy-like proportions.

### Key Characteristics
- **Rendering:** Clean 3D with soft shadows and reflections
- **Objects:** Rounded, friendly 3D shapes (Pixar/Nintendo aesthetic)
- **Text:** Floating 3D text with depth and perspective
- **Lighting:** Soft ambient + directional light
- **Background:** Gradient from light to medium tone

### Color Guidance
- Primary: Module colors with glossy finish
- Shadows: Soft ambient occlusion
- Highlights: Glossy reflections on curved surfaces
- Background: Light gradient matching module colors

### Mood
Premium, modern, playful, friendly, polished

### Tips for Best Results
- Use rounded corners on all 3D objects
- Add subtle reflections on flat surfaces
- Include soft shadow under objects
- Make proportions toy-like (not realistic)
- Use gentle perspective (not extreme)

### Example Use Cases
- Product or tool overviews
- Gamification or rewards lessons
- Modern tech platforms
- Lessons requiring premium feel

## General Guidelines (All Styles)

### Mandatory Elements
1. **Lesson Title:** Clear, prominent, top center
2. **Module Title:** Smaller subtitle below lesson title
3. **Key Concepts:** 3-5 concepts from lesson content
4. **Sinsajo Branding:** Turquoise hummingbird + footer (bottom-right)
5. **Readability:** Text MUST be clearly readable (TOP PRIORITY)

### Aspect Ratio
All summary images: **16:9 landscape** (ideal for slides/presentations)

### Text Legibility
- Minimum font size equivalent: 24px for body text
- High contrast between text and background
- Avoid busy backgrounds behind text
- Use text outlines or backgrounds if needed

### Concept Organization
- Number concepts clearly (1, 2, 3...)
- Use visual hierarchy to emphasize main vs sub-concepts
- Connect related concepts with lines/arrows
- Balance visual weight across composition

### Module Color Integration
Each module has unique palette - ensure style adapts colors while maintaining its core aesthetic.

## Rotation Example: 10-Lesson Module

```
Lesson 0: Chalkboard
Lesson 1: Modern Infographic
Lesson 2: Isometric
Lesson 3: Watercolor
Lesson 4: Comic
Lesson 5: Neon/Dark
Lesson 6: Vintage Notebook
Lesson 7: 3D Render
Lesson 8: Chalkboard (cycle repeats)
Lesson 9: Modern Infographic
```

This ensures students see all 8 styles if the course has 8+ lessons, maintaining visual interest throughout.
