# Sinsajo Creators Branding Guide

Brand guidelines for ZoneKlass course images powered by Sinsajo Creators.

## Brand Identity

**Sinsajo Creators** is the AI image generation and content creation brand behind ZoneKlass educational platform.

**Website:** www.sinsajocreators.com

**Tagline:** "AI-powered educational content creation"

---

## Logo Description

Since the AI model cannot access external files, we describe the logo in detail for prompt inclusion.

### Hummingbird Symbol

**Shape:** Stylized hummingbird in side profile (facing right)

**Pose:** Wings spread upward in dynamic V-shape, suggesting motion and energy

**Style:**
- Simplified silhouette (not photorealistic)
- Smooth curves, geometric simplification
- Recognizable as hummingbird even at small sizes
- Elegant and modern, not cartoonish

**Color:** Turquoise/teal (#06B6D4 or similar cyan-turquoise)

**Symbolism:**
- Agility and precision (hummingbird flight)
- Energy and vitality (rapid wing movement)
- Natural intelligence (pollination, memory)
- Beauty in small details (fitting for AI-generated art)

### Typography

**Brand Name:** "sinsajo creators"

**Style:**
- All lowercase for modern, approachable feel
- Clean sans-serif font
- Thin or regular weight (not bold)
- Spaced evenly below hummingbird symbol

**Color:** Same turquoise as hummingbird (#06B6D4)

## Logo Placement Rules

### Position
**Bottom-right corner** of all generated summary images

### Size
- Small and subtle (approximately 5-8% of image height)
- Must not overpower the educational content
- Should be visible but not distracting

### Spacing
- 20-30px margin from right edge
- 20-30px margin from bottom edge
- Clear space around logo (no overlap with content)

### Composition
```
┌────────────────────────────────┐
│                                │
│    [Educational Content]       │
│                                │
│                                │
│                        🐦      │  ← Hummingbird
│                   sinsajo      │  ← Brand name
│                   creators     │
│         www.sinsajocreators.com│  ← Footer
└────────────────────────────────┘
```

## The Branding Prompt Block

This exact block is included in ALL summary image prompts:

```
BRANDING (bottom-right corner, small and subtle):
- A small turquoise/teal hummingbird silhouette (simple, elegant, side profile with wings spread upward)
- Below the hummingbird: "sinsajo creators" in small text
- Use turquoise color for the hummingbird

FOOTER (bottom center, small neat text):
- "www.sinsajocreators.com"
```

### Why This Format?

1. **Consistent Placement:** Bottom-right ensures no conflict with titles (top-center)
2. **Dual Branding:** Logo + website drives awareness and credibility
3. **Subtle Integration:** Small size respects educational content priority
4. **Professional Look:** Footer text gives polished, official appearance

## Do's and Don'ts

### Do's ✅
- Keep hummingbird in turquoise color family (#06B6D4)
- Maintain side profile view (facing right)
- Use lowercase for "sinsajo creators"
- Place in bottom-right corner
- Keep size small and subtle (5-8% of image height)
- Ensure branding doesn't obscure educational content
- Adapt style to match overall image aesthetic

### Don'ts ❌
- Don't make logo larger than content
- Don't place logo in center or top of image
- Don't use colors other than turquoise for hummingbird
- Don't use uppercase for brand name
- Don't omit the footer URL
- Don't let branding overpower the lesson content
- Don't rotate or flip the hummingbird orientation

---

## Accessibility Considerations

### Contrast
- Ensure hummingbird and text have sufficient contrast with background
- If background is dark, use lighter turquoise or white outline
- If background is light, use standard turquoise

### Readability
- Footer URL must be readable even at small sizes
- Use clean, sans-serif font for maximum legibility
- Avoid decorative fonts that reduce readability

### Positioning
- Never place branding where it obscures key concepts
- Ensure clear visual separation from educational content
- Use adequate padding/margin
